# Jobsy

Jobsy is a LinkedIn‑style web application designed to connect professionals, showcase profiles, and share posts in a clean, modern interface. It was built as a learning and portfolio project to demonstrate front‑end development, responsive design, and interactive features.

## Features

- Profile Section  
  Upload and customize your profile picture, edit name, profession, and bio, and display connection and view stats.

- Post Section  
  Create posts with text and media (articles, images, videos). Posts appear in a feed layout similar to LinkedIn.

- People Section  
  Suggested connections displayed in a sidebar with profile previews.

- Navigation  
  Search bar for companies and locations, icons for connections, saved jobs, inbox, and messages.
